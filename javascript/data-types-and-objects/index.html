Primitive Data Types

Primitive data types is a fundamental data types. Which is built in JavaScript

    There are 7 types of Primitive data types

    Tips to remember it "NN SS BB U"

  1. N = null
  2. N = number
  3. B = boolean
  4. B = bigint
  5. S = string
  6. S = symblo
  7. U = undefined
